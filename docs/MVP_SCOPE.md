# MVP Scope

## In Scope

### Public Website
- Landing page
- How it works page
- Founder CTA
- Builder CTA
- Login page

### Authentication
- Supabase auth
- User role selection
- Approval status

### Founder Experience
- Founder profile
- Project application form
- View submitted projects
- View AI analysis after submission
- View project status
- View assigned team once approved

### Builder Experience
- Builder profile form
- Skills and role selection
- Availability details
- Interests and project goals
- Approval status
- View assigned projects

### Admin Experience
- Admin dashboard
- Review submitted projects
- Review builder profiles
- Approve/reject projects
- Approve/reject builders
- View AI role mapping
- View readiness scores
- View 30-day roadmaps
- Assign builders to projects
- Update project status

### AI Features
- Role mapping
- Readiness score
- 30-day milestone roadmap
- Match suggestions for admins

### Project Tracking
- Project status pipeline
- Assigned team members
- Missing roles
- Basic admin notes
- Simple progress status

## Out of Scope

Do not build these in MVP:

- Real-time chat
- Full project management system
- Kanban boards
- Calendar scheduling
- Payments
- Contract signing
- Equity tracking
- Investor portal
- Public marketplace
- Public project browsing
- Automatic matching without admin approval
- Mentor portal
- University admin portal
- Native mobile app
- Complex notifications
- File storage beyond basic profile links
