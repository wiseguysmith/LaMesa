# MVP Scope

## MVP Goal

The MVP should prove that ISD can run the first Founder 12 cohort through La Mesa.

The platform should help ISD:
- Receive founder applications.
- Evaluate founder and venture readiness.
- Select exactly 12 accepted founders.
- Coordinate accepted-founder access to Founders Coffee, ISD member privileges, and AI sessions.
- Match accepted founders with builders when needed.
- Track movement toward prototype readiness.

## In Scope

### Public Website

- Founder 12 landing page.
- How it works page.
- Founder CTA.
- Builder Network CTA.
- Login page.

### Authentication

- Supabase auth.
- User role selection.
- Approval status.
- Accepted-founder gating.

### Founder Applicant Experience

- Founder 12 application form.
- View submitted application.
- View application status.
- View AI readiness analysis after submission.
- View 30-day roadmap after submission.
- View ISD next steps.

### Accepted Founder Experience

Accepted founders should see a Founder 12 member experience.

MVP modules:
- Founder 12 acceptance status.
- Founders Coffee details and RSVP/status surface.
- ISD member privileges overview.
- AI session information.
- External community chat link, if available.
- Team formation status.
- Assigned builders.
- 30-day roadmap.
- Basic progress updates.

### Builder Experience

- Builder Network profile form.
- Skills and role selection.
- Availability details.
- Interests and project goals.
- Approval status.
- View assigned Founder 12 ventures only.

### Admin Experience

- Cohort Command Center.
- Review submitted Founder 12 applications.
- View AI readiness reports.
- Shortlist founders.
- Accept/reject founders.
- Track accepted founders count against the 12-seat target.
- Review builder profiles.
- Approve/reject builders.
- View AI role mapping.
- View 30-day roadmaps.
- Assign builders to accepted ventures.
- Update venture/cohort status.
- Add internal notes.

### AI Features

- Founder readiness score.
- Founder 12 fit analysis.
- Role mapping.
- 30-day milestone roadmap.
- Match suggestions for admins.

### Venture Tracking

- Founder status pipeline.
- Venture status pipeline.
- Assigned team members.
- Missing roles.
- Basic admin notes.
- Simple progress status.

## Out of Scope

Do not build these in MVP:

- Native real-time chat.
- Full community platform.
- Full project management system.
- Kanban boards.
- Full calendar scheduling.
- Payment processing.
- Contract signing.
- Equity tracking.
- Investor portal.
- Public marketplace.
- Public project browsing.
- Automatic founder acceptance.
- Automatic builder assignment.
- Mentor portal.
- University admin portal.
- Native mobile app.
- Complex notifications.
- File storage beyond basic profile links.
- Full ISD membership CRM.
- Full LMS or course system.

## MVP Restraint

If a feature does not help ISD select, coordinate, support, or move the 12 accepted founders toward prototype readiness, defer it.
